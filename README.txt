schubwerk Analytics Plugin für Wordpress: DSGVO-konformes, serverseitiges Tracking (cookieless)

== Description ==

schubwerk Analytics ist eine DSGVO-konforme, serverseitige Analytics-Lösung zur Ermittlung des Website-Traffics. Mit Hilfe des Plugins lassen sich Besucherzahlen, die Anzahl der stattgefundenen Sitzungen, sowie das Nutzerverhalten erheben und analysieren.
Darüber hinaus bietet das Plugin die Möglichkeit, benutzerdefinierte Formulare zur Steuerung von Leadstrecken und Generierung neuer Leads einzubinden.

Im Gegensatz zu anderen Analytics-Anbietern erfolgt die Erfassung der Daten nicht mit Hilfe von Cookies, sondern serverseitig. Dabei werden keinerlei persönliche Daten erhoben, gespeichert oder verarbeitet. Die Einholung der Zustimmung des Nutzers via Cookie-Banner entfällt. Dadurch ermittelt schubwerk Analytics ein deutlich vollständigeres Bild als Anbieter, deren Analytics-System auf Cookies basiert und deshalb vom Nutzer abgelehnt werden kann. Alle von schubwerk Analytics erhobenen Daten werden auf Servern in Deutschland gespeichert.

== Installation ==

1. Plugin über den Plugin-Store herunterladen
1. Plugin aktivieren
1. In den Plugin-Einstellungen den individuellen Konto-Schlüssel hinterlegen

Sollten Sie noch nicht über ein schubwerk Konto verfügen oder Fragen zu unserem Plugin haben, können Sie gerne ein kostenloses Erstberatungsgespräch vereinbaren.
